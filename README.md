# 🍳 Mes Recettes Odin

Bienvenue ! C'est mon tout premier projet pour **The Odin Project**. L'idée était simple : apprendre à coder en HTML tout en partageant quelques plats qui donnent faim.

### 📖 Au menu :

* **Risotto crémeux aux champignons** : Le réconfort absolu. Un classique italien onctueux.
* **Saumon grillé citron & herbes** : Frais, léger et super rapide à faire. Le genre de plat parfait quand on veut bien manger sans passer 3 heures en cuisine.
* **Poulet rôti à la méditerranéenne** : Ma recette coup de cœur. Un mélange d'épices du Maghreb, des patates fondantes et des olives. C'est le plat familial par excellence.

### 🛠️ Le côté technique
Juste du HTML sémantique pur pour structurer les recettes, les ingrédients et les étapes. J'ai aussi ajouté une couche de CSS pour que l'expérience soit plus agréable visuellement.
